# Toronto Micro-Atlas

Toronto Micro-Atlas is an independent web publication of Field Guides about noticing Toronto. Each guide maps a set of located observations: places where the city's landscape, infrastructure, and topography do something specific and worth recording. The guides are written from direct observation. Sources are cited. Nothing is reviewed; things are described.

Field Guide 01, Hidden Landscapes, is an interactive illustrated map of eight threshold locations across Toronto: the bridges, stair descents, trail entrances, and underpasses where the city's surface cracks open into something older. The eight sites span the Don Valley, the Humber, the Davenport Escarpment, the Rosedale ravine system, the Belt Line corridor, the Beaches, and the Rouge. The map is plotted against City of Toronto Open Data and OSM-derived context; observations are original.

The project is built with Astro, TypeScript, and MapLibre GL JS for the client-side map layer. Deployment target is GitHub Pages. Data sources include City of Toronto Open Data, Toronto and Region Conservation Authority datasets, and OpenStreetMap-derived context. All source attributions are documented in `data/README.md` as they are added.
